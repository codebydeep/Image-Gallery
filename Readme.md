# 🖼️ Image Gallery

A responsive Image Gallery built using HTML, TailwindCSS, and JavaScript, featuring a lightbox with navigation. 

---

## 🚀 Features  

- 🖼️ Responsive image gallery layout.

- 🔍 Lightbox with transitions effects.

- ⬅️➡️ Previous & Next navigation buttons.

- 🎨 Animated background blobs for a modern look.

- 📱 Fully responsive for desktop & mobile.


---

## 🛠️ Tech Stack  
- **HTML5** – Structure  
- **TailwindCSS** – Styling + Background
- **JavaScript** – Core logic
